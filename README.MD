## Безопасность

- Образы собираются с использованием многоэтапной сборки (multi-stage builds)
- Финальные образы запускаются от имени непривилегированного пользователя (non-root)
- Используются минимальные базовые образы (Alpine)
- Docker‑секреты хранятся в GitHub Secrets
- Сканирование уязвимостей с помощью Trivy интегрировано в CI/CD
